# 学生积分管理系统1

基于 Web 的课堂学生积分管理工具，支持学生管理、分组管理、积分规则、Excel 导入导出、数据云端同步。

## 功能

- 用户注册，审批流程
- 学生增删改查，快速加减分
- 分组管理，按组加减分
- 可配置的积分规则模板
- Excel 批量导入/导出
- 数据云端存储（Jsonbin.io），修改器与显示器实时同步（有小范围时差）
- 撤销操作（Ctrl+Z，最多 50 步）
- 排名展示支持语音播报、头像显示
- （密码和用户数据采用加密，提供基础安全性，密码采用PBKDF2 + 哈希，个人数据采用 AES-256-GCM（秘钥随机生成96位）+哈希，个人数据跟密码是密码，意味着密码缺失全部崩盘）
